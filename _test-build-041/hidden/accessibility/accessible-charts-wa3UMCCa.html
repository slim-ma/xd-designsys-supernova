Accessible Charts

When choosing colors for a chart, pair according to the recommended colors from Chart Colors.

Testing for Chart Colors

Did you know that *67% of the current WCAG success criteria can be covered in the design phase?

Using plugins and dev tools, we can easily simulate the appearance of colors of people with color deficiencies.

Recommended Accessibility Tools:

Use the Accessible Colors for Charts table to choose the base colors as a starting point. Perform a test with these recommended Figma plugins:

  • Stark for checking color and text contrasts
  • Color Blind to simulate colors.

Comparison Charts

Use clear patterns along with tints and shades to indicate value intensity from the ordered sequence displayed in Accessible Colors for Charts.

Positive and Negative Values

Use Green for positive, Red for negative, and Gray 200 for neutral values with patterns on each comparison color for better context.

Flex Values

Moody’s Blue and orange can be used to highlight differences or ranges of data that may not reflect decisively positive or negative values.
