Use in Figma

How to use the Moody's Logos component in Figma

From the M’ Core Brand Libray in the Assets panel, drag or simply copy, then paste the Moody’s Logos component into your Figma file to start using.

 

Customize the Moody’s Logo in the properties panel in the Design panel with the following properties:

  • Theme: Dark or Light
  • Division: Default, Moody’s Analytics, or Moody’s Investors Service
  • Show: Default (None), OU, or Tagline
  • End Use: Marketing or Product
  • Size: Default or Minimum
